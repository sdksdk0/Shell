有关某个命令的详细信息，请键入 HELP 命令名
cd            显示当前目录的名称或将其更改。
copy          将至少一个文件复制到另一个位置。
date          显示或设置日期。
del           删除至少一个文件。
dir           显示一个目录中的文件和子目录。
exit          退出 CMD.EXE 程序(命令解释程序)。
help           提供 Windows 命令的帮助信息。
md             创建一个目录。
rd             删除目录。
ren            重命名文件。
rm             删除目录。
tree          以图形方式显示驱动程序或路径的目录结构。
type           显示文本文件的内容。
cat 		查看文件内容
